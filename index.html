Gerenciamento de Estoque

O principal objetivo do site é fazer o gerenciamento de estoque de uma empresa atacadista

Descrição do Sistema

Este sistema permite o controle eficiente dos produtos em estoque, facilitando a gestão e a tomada de decisões.

Com uma interface intuitiva, os usuários podem facilmente navegar pelo sistema e acessar as funcionalidades necessárias para o gerenciamento de estoque.

As principais características do site incluem:

  • Adicionar produtos
  • Editar produtos
  • Remover produtos
  • Visualizar produtos em estoque

Além disso, o sistema oferece recursos adicionais para aprimorar a gestão de estoque, como:

  1. Gerenciar fornecedores
  2. Gerar relatórios de estoque
  3. Configurar alertas de estoque baixo
